# Job Portal — Design System

A component library for the Job Portal recruitment platform. Built with React, TypeScript, and Tailwind CSS v4.

---

## Getting Started

```bash
git clone <repo-url>
cd job_portal/frontend
npm install
npm run dev        # http://localhost:5173
npm run storybook  # http://localhost:6006
```
